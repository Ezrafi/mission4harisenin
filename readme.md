# To-Do List App

A simple and elegant to-do list application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks with priority levels (Low, Medium, High)
- Mark tasks as completed
- Delete individual tasks
- Delete all tasks at once
- Displays current date
- Priority indicators with color-coded borders (High: red, Medium: orange, Low: blue)
- Responsive design

## Project Structure

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Application logic

## How to Use

1. Open `index.html` in your web browser
2. Type a task in the text area
3. Select a priority level from the dropdown
4. Click "Tambah Tugas" to add the task
5. Check the checkbox to mark a task as done
6. Click "Hapus" to delete a specific task
7. Click "Hapus Semua Daftar" to delete all tasks
